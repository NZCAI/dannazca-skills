# EdgarTools MCP for Nazca

This package productizes EdgarTools as a Nazca MCP capability under:

`tools/edgartools-mcp/`

## Key assets
1. `tool-description.md` — description, PRD fit, requirements, roadmap
2. `installation-guide.md` — Nazca setup (stdio default + compatibility + HTTP WIP)
3. `mcp-config.template.json` — ready `mcpServers` template
4. `use-cases.prompts.md` — PE/VC diligence and follow-on prompts
5. `skill-routing.md` — how to combine with Nazca skills

## Additional assets
- `one-prompt-installation-guide.md` — install prompt for Factory + Claude
- `install-from-git.prompt.md` — install prompt for new machines from repo
- `http-wip/` — pilot HTTP structure
